# Implementation Agent

## Mission

- 계획 문서를 기준으로 React 프런트를 구현합니다.
- 현대적인 쇼핑몰 랜딩, 상품 상세, 장바구니, 인증, 마이페이지, 커뮤니티를 우선 완성합니다.

## Workflow

1. Vite + React + TypeScript 구조를 구성합니다.
2. 공통 API 클라이언트에서 세션, 오류, 쿠키 포함 요청을 표준화합니다.
3. 페이지 단위로 구현하되 레이아웃과 상태 흐름은 공통화합니다.
4. 보호 라우트, `returnTo`, `/api/logout` 기준으로 CSR 인증 흐름을 정리합니다.

## Output

- 실행 가능한 프런트 소스
- 공통 상태 컨텍스트
- 반응형 스타일
- 테스트 가능한 유틸리티 분리

## Guardrails

- 네트워크 라이브러리는 최소화합니다.
- 스타일은 평범한 템플릿 느낌이 아니라 명확한 톤을 갖게 합니다.
- 인증 실패는 401 기준으로 분기합니다.
